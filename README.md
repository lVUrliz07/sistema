# Eiser Ecommerce Template

Plantilla frontend para ecommerce desarrollada con HTML, CSS, SCSS y JavaScript.  
Incluye múltiples vistas listas para usar en tiendas online, con diseño responsive y componentes reutilizables.

---

## Características

- Diseño responsive basado en Bootstrap  
- Páginas completas de ecommerce  
- Componentes reutilizables (navbar, footer, cards)  
- Integración de librerías JS para UI/UX  
- Estructura organizada de estilos con SCSS  
- Plantilla lista para personalización  

---

## Estructura del Proyecto
├── css/ # Archivos CSS compilados
├── scss/ # Estilos fuente en SCSS
├── js/ # Scripts principales
├── vendors/ # Librerías externas
├── img/ # Recursos gráficos
├── fonts/ # Tipografías
├── *.html # Vistas del sitio
├── contact_process.php # Procesamiento de formulario

---

## Páginas Incluidas

- `index.html` – Página principal  
- `category.html` – Listado de productos  
- `single-product.html` – Detalle de producto  
- `cart.html` – Carrito de compras  
- `checkout.html` – Proceso de compra  
- `blog.html` – Blog  
- `single-blog.html` – Detalle de blog  
- `tracking.html` – Seguimiento de pedidos  
- `contact.html` – Contacto  

---

## Tecnologías

- HTML5  
- CSS3 / SCSS  
- JavaScript (jQuery)  
- Bootstrap  
- Librerías UI (Owl Carousel, Nice Select, Lightbox, etc.)  

---

## Instalación

1. Clonar el repositorio o descargar los archivos  
2. Abrir `index.html` en el navegador  

No requiere backend ni instalación adicional para su funcionamiento básico.

---

## Personalización

- Modificar estilos en `scss/` y compilar a `css/`  
- Editar contenido directamente en los archivos HTML  
- Reemplazar imágenes en la carpeta `img/`  
- Ajustar scripts en `js/` según necesidades  

---

## Uso

Esta plantilla está diseñada como base para proyectos de ecommerce frontend.  
Puede integrarse con cualquier backend o API.
